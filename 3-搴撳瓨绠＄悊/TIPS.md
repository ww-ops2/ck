# 💡 采购单使用小贴士

## 🚀 快速技巧

### 1. 快速添加多个物品
```
点击 "+" 按钮 → 填写第一行 → 再点击 "+" → 填写第二行
→ 重复直到所有物品添加完毕 → 提交
```

### 2. 智能类别输入
```
方式1: 点击类别框 → 从下拉列表选择
方式2: 直接输入新类别名称 → 自动创建
方式3: 输入部分文字 → 模糊匹配已有类别
```

### 3. 利用历史记录提速
```
第一次: 手动输入品牌/型号
第二次: 点击输入框 → 从历史选择（快50%！）
```

### 4. 金额自动计算
```
只需输入数量和单价 → 金额自动计算
修改任意值 → 金额实时更新
```

## ⚡ 效率提升技巧

### 批量录入法
1. 先连续点击"+"添加所有需要的行数
2. 然后从上到下依次填写
3. 相似物品连续添加，利用历史记录

### 类别预定义
- 提前在"品类管理"中定义常用类别
- 或在首次使用时直接输入，系统自动创建

### 快捷键（未来版本）
- Tab: 切换到下一个输入框
- Enter: 确认并添加新行（计划中）
- Esc: 取消当前操作

## 🔍 常见问题

**Q: 如何删除不需要的行？**  
A: 点击行尾的 ✕ 按钮

**Q: 商品编码能修改吗？**  
A: 不能，SKU编码由系统自动生成以保证唯一性

**Q: 类别输入后没保存？**  
A: 确保输入后点击了其他地方（失焦触发保存）

**Q: 品牌历史怎么清除？**  
A: 目前不支持手动清除，会永久保存
