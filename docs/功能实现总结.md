### ✅ 4. 数据管理

#### 4.1 数据结构设计

**采购单 (Purchase Order)**
```javascript
{
  id, code, purchase_date, purchaser, supplier,
  items: [{name, brand, model, quantity, unit, price, amount}],
  total_amount, status, created_at, remark
}
```

**入库记录 (Stock In Record)**
```javascript
{
  id, code, purchase_order_id, purchase_order_code,
  stockin_date, batch_code,
  items: [{...actual_quantity...}],
  total_quantity, total_amount, status,
  confirmed_by, confirmed_at, remark
}
```

**库存明细 (Inventory)**
```javascript
{
  id, code, name, brand, model, category,
  stock, unit, safety_stock,
  last_stockin_date, last_stockin_batch,
  source, created_at
}
```
